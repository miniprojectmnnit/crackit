# CrackIt 🚀

> **Master your technical interviews with AI-powered realism.**

CrackIt is an advanced interview simulation platform that helps developers prepare for **Technical**, **DSA**, **System Design**, and **HR** rounds using specialized AI agents. It provides a realistic environment with real-time feedback, code editing, and personalized performance reports.

## ✨ Key Features

- 👤 **Resume-Tailored Interviews**: Questions generated based on *your* specific experience.
- 🧩 **DSA Specialist Mode**: Multi-phase coding rounds (Intuition -> Code -> Evaluation).
- 🏗️ **System Design Rounds**: High-level architectural challenges with AI feedback.
- 🤝 **HR & Behavioral Simulation**: Practice the STAR method with realistic follow-ups.
- ⚡ **Real-Time Interaction**: Seamless chat, code, and voice-feedback signals.
- 🔐 **Secure API Key Wallet**: Your LLM keys are AES-256 encrypted and stored safely.

## 🛠️ Quick Start

### 1. Prerequisites
- Node.js (v18+)
- MongoDB Atlas
- Clerk (for Auth)
- LLM API Keys (Gemini, Groq, or OpenAI)

### 2. Setup
Clone the repository and run both client and server:

```bash
# Install dependencies in root
npm install

# Run both Client and Server concurrently
npm run dev
```
